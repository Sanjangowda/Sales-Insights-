# Sales-Insights
## Visualization of a hardware company's sales trends over past 4 years
---
This project visulize the sales data of a company over past 4 years across markets considering following aspects:
- Total Revenue
- Total Sales Quantity
- Revenue by Market
- Sales Qantity by Market
- Top 5 Customers
- Top 5 Products
- Revenue Trend over the years
---

Try this with following steps:
> STEP 1:
- Download the SQL dump file from the repository
- Use SQL workbench for basic data analysis with querying
> STEP 2:
- Connect your SQL databse from SQL Workbench to Power BI
- Import all the tables from the database
> STEP 3:
- Transform the required tables using DAX formulas in Power BI
- Model the data
> STEP 4:
- Use right visualize charts and graphs that suits the data
- Design the dashboard/report in appealing way and can be understood by others
